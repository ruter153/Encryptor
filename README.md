How to install:
  - Download TinyInstaller.exe from the releases tab
  - Run TinyInstaller.exe it will download all needed files

How to use: 

ENCRYPT:
  - Option 1:
    - Right click on the file which you want to encrypt
    - Click on "Encrypt with Encryptor" (if you are using Windows 11 you need to click on "show more options" first to see it)
    - A terminal window will open
    - Type there your password and press Enter
    - Your File will be encrypted
  - Option 2:
    - Open the shortcut on your Desktop
    - Type in the desired encryption mode (encrypt) and press Enter
    - Copy the File path of the file which you want to encrypt, in the explorer and paste it in the terminal window and press Enter
    - It will ask you for your desired password. Just type it in and press Enter
    - Your File will be encrypted

DECRYPT:
  - Option 1:
    - Double tap the encrypted file
    - Type in your password and press Enter
    - Your file will be decrypted as long as the password was correct and the encrypted file was not modified
  - Option 2:
    - Open the shortcut on your Desoktop
    - Type in the desired encryption mode (decrypt) and press Enter
    - Copy the File path of the file which you want to decrypt, in the explorer and paste it in the terminal window and press Enter
    - Type in your password for the file
    - Your file will be decrypted as long as the password was correct and the encrypted file was not modified
